Customer Relationship Management (CRM) System

This project is a Python-based Customer Relationship Management (CRM) application developed to manage customer information, record interactions, track sales, and analyze customer behavior. It uses an SQLite database for data storage and provides a user-friendly graphical interface (GUI) for efficient daily operations.

Key Features:

Customer information management
Customer interaction tracking
Sales and payment management
Customer behavior analysis and scoring
Next purchase prediction
Management reports and analytics
Search and filtering capabilities
Excel data import
